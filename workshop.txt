title=[B41] Map Legend System
description=This mod adds a fully customizable, toggleable legend and information panel to the in-game map. Designed especially for server administrators, it allows you to display event details, symbol explanations and other map-related information in a clean, scalable side window.
description=
description=[h1]How it works[/h1]
description=[list]
description=[*] [b]Toggleable Map Panel:[/b] Adds a dedicated side window directly inside the world map UI.
description=[*] [b]Custom Content Source:[/b] A simple text file stores all legend and information content. If the file does not exist, it is automatically generated with default content.
description=[code]Zomboid/Lua/MapLegendSystem/MapLegendSystem.txt[/code]
description=[*] [b]Versioned updates:[/b] The [i]first line[/i] contains a version string (e.g., [i]VERSION = 1.1[/i]). Increase it after editing to push updates.
description=[*] [b]Server-controlled in Multiplayer:[/b] The server manages and distributes legend content to connected players.
description=[*] [b]Scalable Layout:[/b] Designed to adapt cleanly to different resolutions and UI scales.
description=[/list]
description=
description=[h1]Singleplayer & Server behavior[/h1]
description=[list]
description=[*] [b]Singleplayer:[/b] Reads directly from your local legend file and checks periodically for version changes.
description=[*] [b]Multiplayer:[/b] The server distributes the legend data. Clients sync on connect and receive updates when the version changes.
description=[*] [b]No restart required:[/b] Edit the file, bump the version number, and changes propagate automatically.
description=[/list]
description=
description=[h1]Key features[/h1]
description=[list]
description=[*] [b]Map-integrated UI:[/b] Seamlessly embedded into the world map interface.
description=[*] [b]Admin-friendly:[/b] Perfect for event explanations, safe zones, symbol guides and server briefings.
description=[*] [b]Rich text formatting:[/b] Supports headers, alignment, custom colors, font sizing, and structured layout.
description=[*] [b]Image support:[/b] Optionally reference custom images for visual legends or symbol examples.
description=[*] [b]Lightweight & optimized:[/b] Minimal performance impact.
description=[/list]
description=
description=[h1]Perfect For[/h1]
description=[list]
description=[*] Roleplay servers marking faction territories
description=[*] Event servers explaining objectives and map symbols
description=[*] Hardcore servers outlining safe zones or restricted areas
description=[*] Community projects displaying building plans or area guides
description=[/list]
description=
description=[h1]Options[/h1]
description=Configure these in the [b]Map Legend System[/b] options:
description=[list]
description=[*] Expand/Collapse the Map Legend Panel
description=[*] Panel Opacity
description=[*] UI Scale Adjustment
description=[/list]
description=
description=[h1]Quick start[/h1]
description=[list]
description=[*] Launch the game/server once to auto-create the legend file.
description=[*] Edit content inside:
description=[code]Zomboid/Lua/MapLegendSystem/MapLegendSystem.txt[/code]
description=[*] Bump the first line version (e.g., [i]VERSION = 1.1[/i]) to distribute updates.
description=[/list]
